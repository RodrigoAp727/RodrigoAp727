## 🏆 Projetos em Destaque

### 🌲 Sistema Tree Prime (Gestão Contábil)
Sistema completo de gestão focado na automação de fluxos de trabalho e controle financeiro. A interface foi desenvolvida em tons de cinza para ergonomia visual, mas o grande diferencial é a modelagem de dados sólida para suportar transações precisas e relatórios gerenciais complexos.

Plataforma administrativa desenvolvida para o gerenciamento de alunos e mensalidades. Focada no controle financeiro e cadastro, utilizando arquitetura de dados estruturada para escalar a operação diária da escola de esportes.
